# Build resources

| File | Use |
|------|-----|
| `icon.png` | Source icon (512×512) — generated by `npm run icons` |
| `icon.ico` | Windows (auto-derived by electron-builder from PNG when possible) |
| `icon.icns` | macOS (auto-derived when possible) |

```bash
npm run icons   # scripts/gen-icons.js → build/icon.png
```

`package.json` `build.icon` points at `build/icon.png`.
