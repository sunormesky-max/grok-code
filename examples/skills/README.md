# Example skills

Community skill ideas. Real skills live as:

```text
~/.grok/skills/<name>/SKILL.md
# or project
.grok/skills/<name>/SKILL.md
```

Copy `sample-commit/` into `~/.grok/skills/commit/` or use **Settings → Skills → 新建**.

See [Grok Skills docs](https://grok.x.ai) / `~/.grok/docs/user-guide/08-skills.md` for frontmatter fields.
